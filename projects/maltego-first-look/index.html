Project 02 • Tool Familiarization

Maltego First Look

This project documents my early exposure to Maltego, including first impressions, initial setup, and learning how link analysis tools can support investigative and OSINT workflows.

Maltego
OSINT
Link Analysis
Investigative Workflow

Objective

Gain a working introduction to Maltego and begin understanding how it can be used to map relationships, organize entities, and support open-source investigations.

Why It Matters

Investigations are often about connections, not just isolated facts. Maltego is valuable because it helps visualize those relationships and makes large sets of information easier to analyze.

What I Explored

  • Basic setup and first interaction with the Maltego interface
  • How entities are placed and organized on a graph
  • The role of transforms in expanding available information
  • How visual mapping supports investigative thinking

What I Learned

My first impression of Maltego was that it is less about flashy visuals and more about structure. It encourages an investigator to think in terms of relationships, patterns, and connected data points. Even early exposure made it clear that link analysis is a different way of thinking than simple web searching.

Early Takeaways

  • Entity mapping can reduce investigative chaos
  • Transforms appear powerful but need to be used deliberately
  • The tool makes connections easier to track visually
  • Documentation matters so findings stay organized

Next Steps

  • Run more hands-on transforms
  • Practice with domains, usernames, and public data points
  • Document a small real-world-style investigation workflow
  • Compare Maltego results with manual OSINT methods

Project Status

In progress. This page marks the beginning of my Maltego learning curve and will expand as I become more comfortable using the platform in structured investigative exercises.
